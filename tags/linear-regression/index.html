Linear Regression - Boston Houseprice Data

Today, I will be looking at a Linear Regression Analysis using the Boston House Price dataset from the Sci-Kit Learn Library. Linear Regression allows the examination of the relationship between inputs and a continuous output. In this case, the output refers to the value of the house price. We will be exploring the data to see whether or not we can learn anything about the relationships between the variables, inference, and whether we can build a model that can predict house prices based on some of the inputs.
